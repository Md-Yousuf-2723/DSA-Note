# DSA-Note

Personal notes taken while doing the **Decode C++ with DSA** course by Coding Wallah (CW). These notes are handwritten/drawn using the **Excalidraw** plugin inside **Obsidian**, which lets me mix diagrams, sketches, and code blocks in one canvas.

> ⚠️ These files are `.excalidraw.md` notes. If you open them as plain text/markdown on GitHub, you'll only see raw JSON data — not the actual drawing. To view them properly, you need Obsidian + the Excalidraw plugin, following the steps below.

---

## 📥 1. Install Obsidian

Obsidian is a free markdown-based note-taking app that runs locally on your machine.

1. Go to the official site: https://obsidian.md
2. Download the installer for your OS (Windows / macOS / Linux).
3. Run the installer and open Obsidian.
4. When it asks to create or open a vault, choose **"Open folder as vault"**.

---

## 📂 2. Get this repository onto your computer

**Option A — Clone with Git**
```bash
git clone https://github.com/Md-Yousuf-2723/DSA-Note.git
```

**Option B — Download ZIP**
1. On the GitHub repo page, click the green **Code** button.
2. Select **Download ZIP**.
3. Extract the ZIP file anywhere on your computer.

---

## 🗂️ 3. Open the repo as an Obsidian vault

1. Open Obsidian.
2. Click **File → Open folder as vault** (or select it on the startup screen).
3. Choose the `DSA-Note` folder you cloned/extracted.
4. Obsidian will index the folder and show the notes in the left sidebar, including the `Decode C++ with DSA - CW` folder.

---

## 🎨 4. Install the Excalidraw plugin

The notes won't render correctly without this plugin, since they're `.excalidraw.md` files.

1. Inside Obsidian, go to **Settings** (gear icon, bottom left).
2. Click **Community plugins** → make sure **Restricted mode** is turned **OFF**.
3. Click **Browse**, and search for **"Excalidraw"**.
4. Select the plugin by *Zsolt Viczian* and click **Install**.
5. After installing, click **Enable**.
6. Close Settings.

---

## 🖊️ 5. Open a note

1. In the left file explorer, navigate to `Decode C++ with DSA - CW`.
2. Click on any module note (e.g. `module 10`).
3. It should automatically open in the **Excalidraw canvas view** inside Obsidian, showing the handwritten notes, diagrams, and embedded code blocks.

If it opens as raw text instead of the drawing:
- Right-click the file → **Open with → Excalidraw**, or
- Make sure the plugin is enabled (Step 4).

---

## 🛠️ Tips for viewing smoothly

- Use `Ctrl/Cmd + Scroll` to zoom in/out on the canvas.
- Use the hand tool (🖐️) to pan around large notes.
- The zoom percentage and stats (word count, backlinks) are shown in the bottom bar.
- If diagrams look tiny/huge on first load, hit the **zoom-to-fit** button in the top-right toolbar of the Excalidraw view.
